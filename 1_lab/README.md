# App for rotating and getting inversion of PGM image   
## [(see code)](PNM_editor.cpp)
```
Arguments in through the command line:
lab#.exe <name_of_input_file> <name_of_output_file> <option>
Where <option> is:
0 - Inversion,
1 - Flip horizontal,
2 - Flip vertical,
3 - Rotation of +90,
4 - Rotation of -90.
```
